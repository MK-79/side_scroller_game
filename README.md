# 横スクロールジャンプアクションゲーム
## 概要
名前の通り横スクロールしていく中で、ジャンプして障害物をよけてスコアを稼ぐゲームです。
スクロールは強制で、だんだん速度が速くなり難しくなります。
### 操作方法
* 左コントロール：小ジャンプ
* スペース：通常ジャンプ
## 製作期間2025/12~2026/1
### 説明
このゲームの詳しい説明としては右方向に強制的にスクロールされる中でジャンプ（スペースキー）と小ジャンプ（左コントロールキー）を駆使して障害物（岩）をよけてスコアを稼いでいくゲームです。横スクロールは500スコアごとに速度が上がっていきます。岩は出現する間隔がランダムで大きさもランダムにしています。
