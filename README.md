# 横スクロールジャンプアクションゲーム
## 概要
名前の通り横スクロールしていく中で、ジャンプして障害物をよけてスコアを稼ぐゲームです。
スクロールは強制で、だんだん速度が速くなり難しくなります。
### 操作方法
* 左コントロール：小ジャンプ
* スペース：通常ジャンプ
### 説明
このゲームの詳しい説明としては右方向に強制的にスクロールされる中でジャンプ（スペースキー）と小ジャンプ（左コントロールキー）を駆使して障害物（岩）をよけてスコアを稼いでいくゲームです。横スクロールは500スコアごとに速度が上がっていきます。岩は出現する間隔がランダムで大きさもランダムにしています。ゲームオーバー時にはスペースキーかコントロールキーでやり直すことができます。

### 感想・製作秘話
今回のゲームは冬休みに入ったくらいから作り始めていたが今年の冬は予定が多く、一気に進めることができなかった。そのため空き時間で行っていたが疲れもありモチベーションがわかなかったし、集中もできないという最悪な状態だった。そのため少しの間離れてからやることでいい感じにモチベーションが沸いて進めることができた。今回のことでなかなか進められないときはいったん離れてリフレッシュする事はとても大切だと思った。ゲームの話になるが元々小ジャンプは実装する予定がなかった。しかし、完成してバランス調整しているうちにあった方が楽しそうだしいい感じになりそうと思い、後付けで実装した。実装しようとコードを追加し、ゲームをしたがジャンプをした瞬間フリーズしてゲームが落ちて焦った。それを直そうとしたが原因が分からずAIと一緒に修正していったが1時間くらいかかってしまい苦労した。
**結局インデントミスだった泣**
